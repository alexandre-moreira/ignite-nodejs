# O que é o node.js

* Plataforma open-source que permite a execução da linguagem javascript do lado do servidor.
* v8 + libuv + conjunto de módulos

# O que o Node veio resolver

* Ryan Dahl
* Barra de progesso do flickr

* Módulos próprios
